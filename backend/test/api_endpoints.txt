Club
(
  View:
    No sign in required.
    can view all (ALL GETS).

  Create / Edit / Delete:
    Must be signed in (ALL POSTS).

  Permissions controlled by UserRole scopes:
  EditClubScope:
    0 = NONE
        - Cannot create clubs
        - Cannot edit clubs
        - Cannot delete clubs

    1 = SELF
        - Can create a club ONLY if you are a member of that club
        - Can edit a club ONLY if you are a member of that club
        - Can delete a club ONLY if you are a member of that club

    2 = ALL
        - Can create any club
        - Can edit any club
        - Can delete any club

  "Member" means your PersonID matches one of:
    boardMember1
    boardMember2
    defaultRaceSecretary
)

(
  Have to be logged in 
  View: viewMeetScope
  0 = none
  1 = self (only meets where you are involved)
  2 = all (view all meets)

  0 = none

  The user is the Race Secretary for the meet
  (Meet.RaceSecretary == current user)

  The user is the Judge for the meet
  (Meet.Judge == current user)

(
  must be logged in
  View: viewRaceResultsScope
  0 = none
  1 = self (only race results for dogs you own)
  2 = all (view all race results)

  0 = none
  1 = self (create/edit/delete race results only for dogs you own)
  2 = all (create/edit/delete any race result)

--------------------------------------------------
USER ROLE
(
  Must be logged in
  Controlled by viewUserRoleScope and editUserRoleScope
  Protected roles: ADMIN, PUBLIC (cannot be edited or deleted)
  
  View scope behavior:
    NONE (0) → no access
    SELF (1) → can only see your own role
    ALL (2) → can see all roles

  Edit scope behavior:
    NONE (0) → no edits allowed
    SELF (1) → can edit a role only if the scopes exactly match their own (title can differ, but permissions must be identical)
    ALL (2) → full edit all roles (except protected roles)
)
--------------------------------------------------
