Club
(
  View:
    No sign in required.
    can view all (ALL GETS).

  Create / Edit / Delete:
    Must be signed in (ALL POSTS).

  Permissions controlled by UserRole scopes:
  EditClubScope:
    0 = NONE
        - Cannot create clubs
        - Cannot edit clubs
        - Cannot delete clubs

    1 = SELF
        - Can create a club ONLY if you are a member of that club
        - Can edit a club ONLY if you are a member of that club
        - Can delete a club ONLY if you are a member of that club

    2 = ALL
        - Can create any club
        - Can edit any club
        - Can delete any club

  "Member" means your PersonID matches one of:
    boardMember1
    boardMember2
    defaultRaceSecretary
)

USER ROLE
(
  View:
    No sign in required.
    Can view all user roles (ALL GET endpoints).

  Create / Edit / Delete:
    Must be signed in (ALL POST endpoints).

  Protected roles:
    ADMIN, PUBLIC (cannot be edited or deleted)

  Permissions controlled by UserRole scopes:

    EditUserRoleScope:
      0 = NONE
        Cannot create roles
        Cannot edit roles
        Cannot delete roles

      1 = SELF
        Can create roles ONLY if the scopes exactly match their own role’s scopes
        Can edit roles ONLY if the scopes exactly match their own role’s scopes
        Can delete roles ONLY if the target role’s scopes exactly match their own role’s scopes
        (Title can differ; permissions must be identical)
        (Still cannot edit/delete protected roles)

      2 = ALL
        Can create roles
        Can edit any role (except protected roles)
        Can delete any role (except protected roles)
)
