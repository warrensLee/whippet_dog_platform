(
  Must be signed in.
  Permissions:
  View: viewDogOwnerScope
    0 = none
    1 = self
    2 = all

  Edit: editDogOwnerScope
    0 = none
    1 = self
    2 = all

  SELF rules:
  View:
    You may only view owners for a dog you already own.

  Edit:
    Add owner:
      You may only add yourself as an owner.
      You do not need to already own the dog.

  Delete owner:
    You may only delete owners for a dog you already own.

  Transfer ownership:
    You may only transfer ownership for a dog you already own.
)

USER ROLE
(
  Must be logged in
  Controlled by viewUserRoleScope and editUserRoleScope
  Protected roles: ADMIN, PUBLIC (cannot be edited or deleted)
  
  View scope behavior:
    NONE (0) → no access
    SELF (1) → can only see your own role
    ALL (2) → can see all roles

  Edit scope behavior:
    NONE (0) → no edits allowed
    SELF (1) → can edit a role only if the scopes exactly match their own (title can differ, but permissions must be identical)
    ALL (2) → full edit all roles (except protected roles)
)
